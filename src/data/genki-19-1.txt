Honorific Verbs
Verbi onorifici

# いる / 行く / 来る = いらっしゃる (irregular conjugation いらっしゃいます)
# 食べる / 飲む      = 召し上がる
# くれる            =  くださる (くださいます)
# する              = なさる (なさいます)
# 寝る              = お休みになる
# 見る              = ご覧になる
# 言う              = おっしゃる (おっしゃいます)
# ～ている           = 〜ていらっしゃる (～ていらっしゃいます)
