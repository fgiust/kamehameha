えばいい

# ... ばいいですか - come dovrei fare per ...
# es. 納豆は、どう食べればいいですか？ - Come si dovrebbe mangiare il natto?
#
# ... ばいいかわかりません - non so come dovrei...
# es. どこで チケットを 買えばいいかわかりません - Non so dove dovrei comprare il biglietto

How should I eat natto?
Come dovrei mangiare il natto?
納豆[なっとう]は、どう食[た]べればいいですか{|？}

How should I go to the station?
Come dovrei andare alla stazione?
駅[えき]までどう行[い]けばいいですか{|？}

What time should I wake up?
A che ora dovrei svegliarmi?
何時[なんじ]に起[お]きればいいですか{|？}

Which train should I take?
Quale treno dovrei prendere?
どの電車[でんしゃ]に乗[の]ればいいですか{|？}

How should I write this kanji?
Come dovrei scrivere questo kanji?
この漢字[かんじ]はどう書[か]けばいいですか{|？}

Non so quando dovrei andare.
