この/その/あの/どの + Noun
この/その/あの/どの + 名詞[めいし]

How much is this watch?
この時計[とけい]はいくらですか

That watch is expensive.
その時計[とけい]は高[たか]いです

Which notebook is #name's?
どのノートが#nameさんのですか

This pen is mine.
このペンは私[わたし]のです

That book over there is cheap.
あの本[ほん]は安[やす]いです

Which umbrella is yours?
どの傘[かさ]があなたのですか

This café is very nice.
この喫茶店[きっさてん]はとてもいいです

That dictionary is big.
その辞書[じしょ]は大[おお]きいです

Which shoes are these?
どの靴[くつ]ですか

This hat is not mine.
この帽子[ぼうし]は私[わたし]のではありません
